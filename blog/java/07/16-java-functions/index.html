Hàm, Overloading và Parameter Passing trong Java

Đăng Nguyễn Hải

October 20, 2025

Hàm giúp tổ chức và tái sử dụng code. Overloading và Parameter Passing làm code linh hoạt hơn.

1. Hàm cơ bản

int sum(int a, int b){
  return a + b;
}

2. Overloading

int sum(int a, int b){ return a+b; }
double sum(double a, double b){ return a+b; }

3. Parameter Passing

  • Java sử dụng pass-by-value cho kiểu nguyên thủy

  • Tham chiếu cho object giúp thay đổi nội dung object

4. Trải nghiệm học

  • Viết nhiều hàm khác nhau → Feynman: “Hàm = robot thực hiện nhiệm vụ”

  • Thử Overloading → ghi nhớ sự khác biệt

5. Kết luận

  • Hàm & Overloading giúp tổ chức code hiệu quả
