Làm việc với Object và Array trong JavaScript

JavaScript Object Array

Trong JavaScript, Object và Array là 2 kiểu dữ liệu “linh hồn” của ngôn ngữ.

1. Object

let student = {
  name: "Đăng",
  age: 21,
  study: function() {
    console.log(`${this.name} đang học JavaScript`);
  }
};
student.study();

2. Array

let scores = [8, 9, 10];
scores.push(7);
console.log(scores);

3. Lặp qua mảng

scores.forEach(s => console.log(`Điểm: ${s}`));

💡 Feynman Tip: Object là “tủ hồ sơ” (có nhãn – giá trị), còn Array là “danh sách số thứ tự”.

Toán tử và biểu thức trong JavaScript

JavaScript Toán tử Biểu thức

Trong JavaScript, toán tử (operator) là ký hiệu thực hiện phép tính hoặc thao tác trên dữ liệu.

1. Toán tử cơ bản

let a = 5, b = 2;
console.log(a + b); // 7
console.log(a * b); // 10
console.log(a / b); // 2.5

2. Toán tử so sánh


console.log(5 == "5");  // true (so sánh giá trị)
console.log(5 === "5"); // false (so sánh giá trị + kiểu)
💡 Feynman Tip: Hãy nghĩ “==” như so sánh bằng mắt, còn “===” như so sánh bằng kính hiển vi (so cả chi tiết kiểu dữ liệu).

3. Toán tử logic


let x = true, y = false;
console.log(x && y); // false
console.log(x || y); // true
console.log(!x);     // false

Câu điều kiện và vòng lặp trong JavaScript

JavaScript Điều kiện Vòng lặp

Câu điều kiện và vòng lặp giúp máy tính ra quyết định và lặp lại công việc tự động.

1. If…else

let score = 80;
if (score >= 85) console.log("Xuất sắc");
else if (score >= 65) console.log("Khá");
else console.log("Cần cố gắng");

2. Switch


let day = 3;

Toán tử và biểu thức trong JavaScript

JavaScript Toán tử Biểu thức

Trong JavaScript, toán tử (operator) là ký hiệu thực hiện phép tính hoặc thao tác trên dữ liệu.

1. Toán tử cơ bản

let a = 5, b = 2;
console.log(a + b); // 7
console.log(a * b); // 10
console.log(a / b); // 2.5

2. Toán tử so sánh


console.log(5 == "5");  // true (so sánh giá trị)
console.log(5 === "5"); // false (so sánh giá trị + kiểu)
💡 Feynman Tip: Hãy nghĩ “==” như so sánh bằng mắt, còn “===” như so sánh bằng kính hiển vi (so cả chi tiết kiểu dữ liệu).

3. Toán tử logic


let x = true, y = false;
console.log(x && y); // false
console.log(x || y); // true
console.log(!x);     // false

Câu điều kiện và vòng lặp trong JavaScript

JavaScript Điều kiện Vòng lặp

Câu điều kiện và vòng lặp giúp máy tính ra quyết định và lặp lại công việc tự động.

1. If…else

let score = 80;
if (score >= 85) console.log("Xuất sắc");
else if (score >= 65) console.log("Khá");
else console.log("Cần cố gắng");

2. Switch


let day = 3;
switch(day) {
  case 1: console.log("Thứ hai"); break;
  case 7: console.log("Chủ nhật"); break;
  default: console.log("Một ngày khác");
}

3. Vòng lặp for, while


for (let i = 1; i <= 5; i++) console.log(i);
javascript
Sao chép mã
let n = 1;
while (n <= 5) {
  console.log(n);
  n++;
}

💡 Feynman Tip: Hãy tưởng tượng for loop là người làm việc theo kế hoạch có sẵn, còn while là người làm đến khi “xong việc”.
