  <h2 style="color: #FF5722; border-bottom: 3px solid #FFCCBC; padding-bottom: 10px; margin-top: 50px; font-weight: 800; font-size: 2.2em; text-align: center;">
        🧠 Học Java — Hành trình cần cả tư duy và sự kiên nhẫn
    </h2>
    <div style="font-size: 1.3em; line-height: 1.8; margin-top: 30px; margin-bottom: 40px; text-align: justify;">
        <p>
            Bắt đầu học **Java** có thể khiến nhiều người cảm thấy choáng ngợp — cú pháp dài, khái niệm “class”, “object”, “method” nghe có vẻ khô khan.
        </p>
        <p style="padding: 10px; border-left: 5px solid #007bff; background-color: #f4f8ff; margin: 20px 0;">
            Nhưng khi đi đúng lộ trình, bạn sẽ thấy Java là **ngôn ngữ cực kỳ tuyệt vời** để xây dựng nền tảng lập trình vững chắc và chuyên nghiệp.
        </p>
        <p>
            Học Java không chỉ là học cú pháp, mà là học **cách tư duy logic và thiết kế phần mềm**. Mỗi dòng code bạn viết ra là một bước tiến trong cách bạn suy nghĩ có hệ thống hơn.
        </p>
    </div>

  <hr style="border: 0; height: 1px; background-color: #ddd; margin: 40px 0;">

  <h2 style="color: #4CAF50; border-bottom: 3px solid #e8f5e9; padding-bottom: 10px; margin-top: 40px; font-weight: 700; font-size: 1.8em;">
        📘 Học lý thuyết đi đôi với thực hành
    </h2>
    <div style="font-size: 1.25em; line-height: 1.8; margin-bottom: 30px; text-align: justify;">
        <p>
            Lý thuyết là nền móng, nhưng **thực hành mới là chìa khóa để bạn nhớ lâu và hiểu sâu**.
        </p>
        <p style="font-style: italic; color: #555;">
            Nhiều người học Java nhanh nản vì chỉ đọc mà không viết — hãy nhớ, *code là ngôn ngữ, phải “nói” nhiều mới giỏi được.*
        </p>
    </div>

  <h3 style="color: #007bff; font-weight: 700; font-size: 1.5em; margin-bottom: 20px;">
        🔹 Cách học hiệu quả:
    </h3>
    <ol style="font-size: 1.25em; line-height: 2.0; padding-left: 20px;">
        <li><span style="font-weight: 600;">Học ít nhưng chắc:</span> Mỗi ngày chỉ cần nắm một khái niệm, nhưng hãy hiểu thật kỹ và viết code thử.</li>
        <li><span style="font-weight: 600;">Code lại sau mỗi bài học:</span> Đừng copy — hãy tự gõ, vì não bạn sẽ ghi nhớ qua hành động.</li>
        <li><span style="font-weight: 600;">Hiểu trước khi chạy:</span> Nghĩ xem chương trình cần làm gì, đầu vào – đầu ra ra sao, rồi mới code.</li>
        <li><span style="font-weight: 600;">Học từ lỗi:</span> Đọc kỹ thông báo lỗi, tra Google, hỏi cộng đồng — mỗi lỗi là một bài học quý giá.</li>

  <h2 style="color: #ff9800; border-bottom: 3px solid #fff3e0; padding-bottom: 10px; margin-top: 40px; font-weight: 700; font-size: 1.8em;">
        💪 Gợi ý bài tập & ví dụ thực tế
    </h2>
    <p style="font-size: 1.25em; line-height: 1.8; margin-bottom: 30px; font-style: italic;">
        Đừng chỉ đọc – hãy **tự tạo ra sản phẩm** dù là nhỏ nhất.
    </p>
    <ul style="font-size: 1.25em; line-height: 2.0; padding-left: 20px;">
        <li>🎓 <span style="font-weight: 600; color: #ff9800;">Quản lý sinh viên:</span> Nhập thông tin, tính điểm trung bình, phân loại học lực.</li>
        <li>💳 <span style="font-weight: 600; color: #ff9800;">Mô phỏng máy ATM:</span> Đăng nhập, rút tiền, chuyển khoản, in sao kê.</li>
        <li>🎮 <span style="font-weight: 600; color: #ff9800;">Mini Game:</span> Viết game đoán số hoặc “Kéo – Búa – Bao”.</li>
        <li>💰 <span style="font-weight: 600; color: #ff9800;">Quản lý chi tiêu:</span> Ghi chép, tổng hợp và hiển thị báo cáo chi tiêu.</li>
        <li>📂 <span style="font-weight: 600; color: #ff9800;">Quản lý file:</span> Lưu và đọc dữ liệu từ file `.txt` để hiểu về I/O trong Java.</li>
    </ul>
    <p style="font-size: 1.2em; line-height: 1.8; margin-top: 20px; font-style: italic; color: #666;">
        Những dự án nhỏ này giúp bạn học Java một cách tự nhiên – **mỗi dòng code là một trải nghiệm thực tế.**
    </p>
